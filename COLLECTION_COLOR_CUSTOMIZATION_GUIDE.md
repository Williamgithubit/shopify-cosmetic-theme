# Collection Page Color Customization Guide
## How to Edit Colors in the Theme Editor

You can now customize all text colors, backgrounds, and accent colors on your collection page directly from the Shopify Theme Editor without touching any code!

---

## üé® Available Color Settings

### **Collection Hero Section**

Navigate to: **Theme Editor ‚Üí Collection Page ‚Üí Collection Hero**

**Color Options:**
1. **Background color** - Main hero section background (Default: #F5EDE6)
2. **Text color** - Title, description, and breadcrumb text (Default: #2C2C2C)
3. **Accent color** - Product count text (Default: #D4A373)
4. **Promo bar background** - Top promotional bar background (Default: #D4A373)
5. **Promo bar text color** - Promotional bar text (Default: #FFFFFF)

**What These Control:**
- Hero section background
- "Products" title color
- Collection description text
- Breadcrumb links (Home / Products)
- "10 products" count text
- "Free shipping" promo bar

---

### **Collection Filters Section**

Navigate to: **Theme Editor ‚Üí Collection Page ‚Üí Collection Filters**

**Color Options:**
1. **Background color** - Main filter area background (Default: #FFFFFF)
2. **Sidebar background color** - Filter sidebar panel (Default: #FDFBF9)
3. **Text color** - All filter labels and text (Default: #2C2C2C)
4. **Accent color** - Buttons, checkboxes, active states (Default: #D4A373)
5. **Border color** - All borders and dividers (Default: #E8DDD4)

**What These Control:**
- Filter sidebar background
- "Filter" heading text
- Filter group labels (Availability, Price, etc.)
- Checkbox fill color when selected
- "Apply" button background
- Active filter tags
- Input field borders
- Sort dropdown styling

## üìù How to Customize Colors

### **Step 1: Open Theme Editor**
1. Go to **Shopify Admin ‚Üí Online Store ‚Üí Themes**
2. Click **Customize** on your active theme
3. Navigate to any **Collection page** (e.g., /collections/all)

### **Step 2: Select Section**
1. Click on the **Collection Hero** section at the top
2. Scroll down to find the **"Colors"** section
3. Click on any color picker to change colors

### **Step 3: Adjust Colors**
- Click the color swatch to open the color picker
- Choose from:
  - **Color wheel** - Pick any custom color
  - **Hex input** - Enter specific hex codes
  - **Recent colors** - Use previously selected colors
  - **Theme colors** - Use your theme's color palette

### **Step 4: Repeat for Filters**
1. Click on the **Collection Filters** section
2. Scroll to **"Colors"** section
3. Customize filter colors

### **Step 5: Save**
Click **Save** in the top right corner

## ‚ö†Ô∏è Best Practices

### **Contrast Requirements (WCAG AA)**
- Text on background: Minimum 4.5:1 contrast ratio
- Large text (18px+): Minimum 3:1 contrast ratio
- Use tools like [WebAIM Contrast Checker](https://webaim.org/resources/contrastchecker/)

### **Color Harmony Tips**
1. **Use 2-3 main colors** - Background, text, and one accent
2. **Keep it consistent** - Use the same accent color throughout
3. **Test on mobile** - Colors may look different on small screens
4. **Consider your brand** - Match your logo and brand colors

### **Common Mistakes to Avoid**
- ‚ùå Low contrast text (hard to read)
- ‚ùå Too many different colors
- ‚ùå Bright backgrounds with bright text
- ‚ùå Neon colors for large areas
- ‚úÖ High contrast for readability
- ‚úÖ Subtle backgrounds
- ‚úÖ Bold accents for CTAs
